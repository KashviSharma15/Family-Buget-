# 📊 Personal Financial Dashboard using Microsoft Excel

## Overview

This project is an interactive Personal Financial Dashboard built using Microsoft Excel. It helps users monitor income, expenses, savings, and spending habits through dynamic charts, Pivot Tables, and KPI cards.

---

## Features

- Monthly Budget Tracking
- Expense Categorization
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Budget vs Actual Analysis
- Savings Analysis
- Expense Breakdown
- Financial Insights
